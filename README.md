## Installation
When in your home directory, run the following:
```bash
mkdir -p .icons/default
cd .icons
git clone https://github.com/SimonCantCode/pink-heart-cursors.git
echo -e "[Icon Theme]\nInherits=pink-heart-cursors" > default/index.theme
```
You can also copy it to a file and run as a script.
